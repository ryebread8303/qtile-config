# QTile configuration

## Groups

I've setup the following groups to automatically grap apps

- Group 1 gets my browser
- Group 2 gets Steam
- Group 3 gets the terminal

## Autostart

I'm starting the WiFi and Bluetooth systray applets by running autorun.sh.
